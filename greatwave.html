Minimal Great Wave

This piece was one of my more complicated projects in Illustrator where I had to use multiple lines with the pen tool. I wanted to create a very simplified version of Katsushika Hokusai's Great Wave to test my knowledge of how to use the pen tool. Hokusai is an inspiration so I wanted to use his famous Great Wave as a way to test myself. I created this with the intent to learn more about the pen tool and how to consistently use it in my projects.
